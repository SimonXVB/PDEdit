### PDEdit is a simple PDF viewer with basic editing, made using Vite/React + TS
Available at: https://pdedit.simondev.xyz/
# Features
- Add PDF Documents
- Remove PDF Pages
- Rotate PDF Pages
- Rearrange PDF Pages
- Download the finished PDF
- Works on mobile devices
